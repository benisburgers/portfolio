About

Beni is a University of British Columbia graduate, outdoor enthusiast, and aspiring web developer. Since graduating, he has worked as a rafting guide, ski instructor, and hiking guide in the Swiss Alps and the Canadian Rocky Mountains. When he's not spending time outside, Beni has been learning the essentials of web development.

Over the last year, Beni has written countless lines of code, learning how to use HTML, CSS, Javascript, Bootstrap, Python, and Git. Currently, Beni is learning how to use Ruby. Next, Beni plans to learn about frameworks such as Ruby on Rails and PHP, and hopes to complete an internship as a web developer.

Calculator

I wrote this calculator with pure javascript, HTML, and CSS. It is mobile friendly. I made sure that the digits overflowing the display are not cut mid-character, but instead are cut in-between characters. I did so by determining the width of the display, the width of the characters, and then limiting the number of characters displayed. I'm especially proud of this feature.

Etch-a-sketch

I wrote this etch-a-sketch program purely with CSS, Javascript, and HTML. Creating a grid with a dynamic number of rows and columns was difficult to achieve purely with Javascript. However, I succeeded. Additionaly, you will notice that the brightness of the individual boxes decreases as you re-enter a box, until it's completely dark. I achieved this effect by using HCS/HCL instead of RGB.
